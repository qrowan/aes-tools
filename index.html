이게 뭐 하는 거?

Mac mini의 Secure Enclave (T2/Apple Silicon 칩) 안에서 생성된 P-256 EC 키쌍. 비밀키는 SE 칩 밖으로 나오지 않음. 디스크의 ~/.config/se-keys/default.bin은 SE에 바인딩된 wrapped reference라 다른 맥에서는 무용지물.

이 페이지는 SE 공개키만 박혀 있음. 입력한 secret을 ECIES (ECDH + HKDF-SHA256 + AES-256-GCM) 로 암호화 → blob. 비밀키 가진 SE만 복호화 가능.

완성 트리거는 /secret <blob> 형태. 텔레그램으로 여름이에게 보내면 즉시 복호화 후 현재 작업 맥락에 사용. 복호화된 내용은 그 시점부터 conversation jsonl에 들어감.

안 돼요/디버그

1. 상단 진단 패널에서 secureContext=true, ✓ crypto.subtle 확인.

2. 페이지가 HTTPS인지 (qrowan.github.io).

3. 안 되면 진단 영역 스크린샷 + 에러 메시지를 여름이에게.
