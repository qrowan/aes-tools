🔐 mini-reboot blob 생성기

전부 브라우저 안에서 처리. 네트워크 전송 없음.

진단 중...
기본값은 재부팅용. 다른 secret 보낼 땐 비우거나 "API key:" 같은 라벨로 변경.
⏳ 암호화 중...
이게 뭐 하는 거?

OpenSSL과 호환되는 형식으로 평문을 암호화:

openssl enc -aes-256-cbc -salt -pbkdf2 -iter 100000 -base64 -A -pass file:<secret>

출력은 매번 달라요 (랜덤 salt 때문). 같은 입력이어도 매번 새 blob 생성 가능, 모두 유효합니다.

Mac mini의 decrypt-blob.sh / mini-reboot.sh가 같은 secret으로 복호화.

안 돼요/디버그

1. Safari/Chrome에서 직접 열어야 함 (Telegram in-app 브라우저는 Web Crypto API 제한될 수 있음). iOS: 파일 → "공유" → "Safari에서 열기".

2. 위쪽 "진단" 영역에 빨간 표시 있으면 거기에 원인.

3. 안 되면 화면 스크린샷 + 진단 영역 메시지를 여름이에게 보내주세요.
